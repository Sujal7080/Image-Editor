# Image Editor

A minimal, modern web-based image editor with real-time filters and preset effects.

## Features

- **Real-time Image Filtering** - Apply filters instantly with smooth sliders
- **9 Filter Controls** - Brightness, contrast, saturation, hue rotation, blur, grayscale, sepia, opacity, and invert
- **Preset Effects** - 9 pre-configured styles including vintage, cinematic, black & white, and more
- **Download Edited Images** - Save your edited images as PNG files
- **Clean Minimal UI** - Modern design with smooth rounded edges and Satoshi font

## Usage

1. Open `index.html` in your browser
2. Click "Choose Image" to upload an image
3. Adjust filters using the sliders on the right panel
4. Try preset effects for quick styling
5. Click "Download" to save your edited image
6. Use "Reset" to restore default filter values

## Technologies

- HTML5 Canvas for image manipulation
- CSS3 with custom properties for theming
- Vanilla JavaScript for filter logic

## Browser Support

Works on all modern browsers that support HTML5 Canvas and CSS custom properties.
